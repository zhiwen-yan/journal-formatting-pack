# Taylor & Francis Official Guidelines

Last checked: 2026-07-06.

Use these official sources first:

- Formatting and templates: https://authorservices.taylorandfrancis.com/publishing-your-research/writing-your-paper/formatting-and-templates/
- Data sharing: https://authorservices.taylorandfrancis.com/publishing-your-research/writing-your-paper/data-sharing/
- Journal-specific instructions pattern: search the journal page on https://www.tandfonline.com/ and open "Instructions for authors".

## What To Verify Each Time

- Target journal Instructions for Authors.
- Whether format-free submission is accepted.
- Article type and word limits.
- Abstract and keywords.
- Reference style.
- Figures, tables, permissions, and supplementary material.
- Disclosure statement, funding, data availability, ethics, consent, and reporting guideline requirements.

## Output Convention

When only publisher-level guidance is found, mark journal-level details as `needs target journal confirmation`.
