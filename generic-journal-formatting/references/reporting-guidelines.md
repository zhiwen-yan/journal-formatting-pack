# Reporting Guideline Source Map

Last checked: 2026-07-06.

Use this map when no target journal is known or when the target journal asks for reporting guideline compliance.

## Common Guidelines

| Study type | Guideline | Official source |
|---|---|---|
| Randomized trial | CONSORT | https://www.consort-spirit.org/ |
| Observational study | STROBE | https://www.strobe-statement.org/ |
| Systematic review or meta-analysis | PRISMA | https://www.prisma-statement.org/ |
| Case report | CARE | https://www.care-statement.org/ |
| Animal study | ARRIVE | https://arriveguidelines.org/ |
| Prediction model | TRIPOD | https://www.tripod-statement.org/ |
| Health economic evaluation | CHEERS | https://www.equator-network.org/reporting-guidelines/cheers/ |
| Qualitative interview/focus group | COREQ | https://www.equator-network.org/reporting-guidelines/coreq/ |
| Routinely collected health data | RECORD | https://www.record-statement.org/ |
| Guideline search hub | EQUATOR Network | https://www.equator-network.org/reporting-guidelines/ |

## Use Rules

- Select reporting guidelines from the study design, not from the journal name alone.
- Ask for the study design if it is unclear.
- Do not claim compliance unless the manuscript contains the required information.
- If a checklist is incomplete, list missing items explicitly.
