# PLOS Official Guidelines

Last checked: 2026-07-06.

Use these official sources first:

- PLOS ONE submission guidelines: https://journals.plos.org/plosone/s/submission-guidelines
- PLOS how to submit: https://plos.org/resource/how-to-submit/
- PLOS ONE data availability: https://journals.plos.org/plosone/s/data-availability
- PLOS ONE figures: https://journals.plos.org/plosone/s/figures

## What To Verify Each Time

- Target PLOS journal and article type.
- Title, authors, affiliations, abstract, and any author summary.
- Data availability statement.
- Funding statement.
- Competing interests.
- Ethics approval, consent, permissions, and trial registration.
- Reporting guideline checklist.
- Reference style.
- Figure, table, supporting information, and file requirements.

## PLOS-Sensitive Items

PLOS often requires transparent data availability and detailed competing interests/funding disclosures. If repository links, accession numbers, restrictions, or ethics details are missing, mark them missing and ask the user to provide the facts.
