# Publisher Routing And Source Map

Last checked: 2026-07-06.

Use this file to identify the likely publisher-specific formatting skill and the first official pages to verify. Treat every URL as a starting point; final rules must come from the current target journal instructions.

| User clue | Route to |
|---|---|

| MDPI, mdpi.com, Nutrients, Foods, IJERPH, JCM | `mdpi-formatting` |
