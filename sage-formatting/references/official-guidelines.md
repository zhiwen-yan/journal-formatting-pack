# SAGE Official Guidelines

Last checked: 2026-07-06.

Use these official sources first:

- SAGE manuscript submission guidelines portal: https://us.sagepub.com/en-us/nam/manuscript-submission-guidelines
- SAGE journals platform: https://journals.sagepub.com/
- Journal-specific author instructions pattern: search the exact journal title plus "SAGE author instructions".

## What To Verify Each Time

- Target journal Manuscript Submission Guidelines.
- Article type and word limits.
- Abstract type and keywords.
- Blinded review and separate title page requirements.
- Reference style.
- Figures, tables, file naming, permissions, and supplementary files.
- Research ethics, informed consent, trial registration, funding, conflicts of interest, data availability, acknowledgments, and reporting guidelines.

## Output Convention

Use SAGE publisher pages only for navigation. Final formatting decisions must cite the target journal's Manuscript Submission Guidelines.
