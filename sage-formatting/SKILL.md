# SAGE Formatting

## Workflow

1. Confirm the exact SAGE journal and article type.
2. Read the journal-specific Manuscript Submission Guidelines. Use `references/official-guidelines.md` as a source map.
3. Check whether the journal requires blinded files, a separate title page, particular reference style, or article-type limits.
4. Audit manuscript structure, references, declarations, figures, tables, and supplementary files.
5. Produce the requested checklist or formatted sections.

## Required Inputs

Ask for:

- Target SAGE journal.
- Article type.
- Manuscript Submission Guidelines URL.
- Manuscript file or text.
- Reference style, if known.
- Human/animal/public data/clinical trial/AI/supplementary material status.

## Checks

Check:

- Title page and anonymization requirements.
- Abstract type, keywords, and article length.
- Main text organization.
- Reference style and citation consistency.
- Figures, tables, file naming, captions, and permissions.
- Research ethics, informed consent, clinical trial registration, funding, competing interests, data availability, acknowledgments, and AI-use disclosure.

## Guardrails

SAGE journals differ widely. Do not transfer limits or style rules from one SAGE journal to another unless the target guidelines explicitly match.
