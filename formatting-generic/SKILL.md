# Generic Journal Formatting

## Purpose

Use this fallback when the target journal or publisher is unknown. Prepare a clean, conservative submission-ready manuscript structure and clearly mark all items that require target-journal confirmation.

## Workflow

1. Ask for the target journal if the user has one.
2. If no journal is known, apply a conservative biomedical/SCI checklist.
3. Keep all journal-specific decisions marked as "confirm with target journal".
4. Preserve scientific meaning and all factual content.
5. Produce the requested checklist, declarations, title page, cover letter, response letter, or revised structure.

## General Structure

For original research, use:

1. Title page.
2. Abstract.
3. Keywords.
4. Introduction.
5. Materials and Methods or Methods.
6. Results.
7. Discussion.
8. Conclusions if appropriate.
9. Declarations.
10. References.
11. Tables.
12. Figure legends.
13. Supplementary material.

For reviews, systematic reviews, meta-analyses, case reports, clinical trials, and animal studies, apply the relevant reporting guideline and ask for the target journal before final formatting.

## Universal Declarations

Check whether these are needed:

- Ethics approval.
- Consent to participate.
- Consent for publication.
- Clinical trial registration.
- Animal ethics approval.
- Data availability.
- Code availability.
- Author contributions.
- Funding.
- Conflicts of interest.
- Acknowledgments.
- AI-use disclosure.
- Supplementary material.

## Reporting Guidelines

Apply likely guidelines:

- CONSORT for randomized trials.
- STROBE for observational studies.
- PRISMA for systematic reviews and meta-analyses.
- CARE for case reports.
- ARRIVE for animal studies.
- TRIPOD for prediction models.
- RECORD for routinely collected health data.
- COREQ for qualitative interviews and focus groups.
- CHEERS for health economic evaluations.

## Guardrails

Never invent missing manuscript facts. Use placeholders and ask the user to confirm. Do not enforce word limits, reference style, abstract type, or figure/table limits until the target journal is known.
